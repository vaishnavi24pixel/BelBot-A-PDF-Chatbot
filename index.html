Instructions
1. Upload one or more PDF files using the file input above.
2. Click on any PDF page to toggle its selection (glowing border) for a section summary.
3. Ask a question to get relevant answers from the PDFs.
4. Click "More" to see additional answers if available.
5. Click "Voice Input" to ask a question using your microphone.
6. Click "Summarize All" to generate a summary of the entire document.
7. Click "Summarize Selection" to generate a focused summary of the selected pages.
8. Visual summaries: Pages corresponding to key summary sentences will glow with an accent dashed border.
